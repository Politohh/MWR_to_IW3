## Features

* Replaces a lot of assets/models from base game
* New vehicles
* New grass, flowers
* New bomb briefcase + bombsite 
* Replaces all signs
* New trees
* Works with old demos
* High quality

## Installation

1. Download.
2. Extract and put the mod.ff and the *.iwd* archive in your cod4 mod directory.
3. Done!
